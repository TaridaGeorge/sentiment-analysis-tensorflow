filmul este un movie.one foarte bun dintre cele mai bune din filmele yash raj. directia este incredibila. scenariul este stralucitor. povestea este excelenta. povesteste despre rahul, care este obosit de kiran, colegul sau, prieten. psiho face lucruri ca vorbind cu mama lui pe un telefon (oricum ea a murit 15 ani inapoi) etckiran este angajat la sunil.rahul face totul ca el sa poata obtine ea.ei chiar incearca sa-l omoare pe sunil, dar el supravietuieste. la locul unde se indreapta spre luna de miere. filmul este nesaturat. sahrukh este superb, juhi este destul de bun, sunny este in medie, anupham este bine si asa este tanvi, dalip a facut bine. filmul apartine lui srk. dialogurile sunt stralucitoare (cele shahrukh si o multime, daca nu cumulat si comedie). "jaadu teri nazar" si "tu mere samne" sunt melodii absolut melodioase.
